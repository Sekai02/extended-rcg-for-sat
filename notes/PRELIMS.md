- Notación y conceptos básicos de teoría de lenguajes:
alfabeto, cadena, lenguaje, cadena vacía, longitud de cadena, concatenación, notación para conjuntos de terminales, no terminales y variables, notación fija para fórmulas booleanas, variables, cláusulas, literales, cadenas y gramáticas.

variable booleana, literal, cláusula, fórmula booleana, interpretación, satisfacibilidad, forma normal conjuntiva (CNF/FNC), tamaño de una instancia, variantes de SAT que usarás como referencia, al menos 2-SAT.

presentación intuitiva de los elementos de las RCG, definición de rango, predicado, aridad, cláusula, sustitución de rango, definición formal de RCG, símbolo inicial, proceso de derivación, reconocimiento, definición de cuándo una cadena o vector de cadenas es reconocido, definición de SRCG como caso particular.

- Propiedades y limitaciones de las RCG relevantes para la tesis:
poder expresivo de las RCG, propiedades de clausura que realmente usarás, complejidad conocida del problema de la palabra en RCG, existencia de casos no polinomiales, diferencias básicas entre RCG y SRCG, limitaciones de las RCG para representar SAT de forma directa y eficiente.
